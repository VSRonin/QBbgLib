void QBbgLib::QBbgWorker::setRequest ( const QBbgRequest &  a)

Sets the request that has to be sent to Bloomberg.

This sets the request that should be sent to Bloomberg, the individual data requests will be split automatically in the most efficient way and sent.
If a request was already sent and is still pending answer from Bloomberg, this method will do nothing.
